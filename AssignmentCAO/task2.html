What is Stack?

A stack is an abstract data type that serves as a collection of elements, with two main principal operations: Push, which adds an element to the collection, and Pop, which removes the most recently added element that was not yet removed.

Switch the CPU into LongMode to switch the kernel into 64bit Mode.

What is Long Mode?

In the x86-64 computer architecture, long mode is the mode where a 64-bit operating system can access 64-bit instructions and registers. 64-bit programs are run in a sub-mode called 64-bit mode, while 32-bit programs and 16-bit protected mode programs are executed in a sub-mode called compatibility mode.

Paging allows us to enter submode which is implemented in main.asm

What is paging?

In computer operating systems, memory paging is a memory management scheme by which a computer stores and retrieves data from secondary storage for use in main memory (virtual memory concept). In this scheme, the operating system retrieves data from secondary storage in same-size blocks called pages.
